# 📊 ExpenseTracker

**ExpenseTracker** is a **high-performance, mobile-first financial management application** built with **React Native and Expo**.  
It helps users **track income and expenses, manage monthly budgets, and visualize spending habits through real-time analytics**.

---
## 🚀 Key Features

- **📷 AI-Powered OCR Receipt Scanner**  
  Scan paper receipts instantly using the device camera or photo library. Powered by **OCR.space** with custom, resilient parsing algorithms that extract the store name, transaction date, and actual total paid (smartly bypassing cash/change lines) while automatically assigning categories.

- **⚡ Real-time Synchronization**  
  Powered by **Firebase Firestore** for instant transaction updates across devices.

- **🔐 Persistent Authentication**  
  Secure login and signup using **Firebase Authentication** with **AsyncStorage** session persistence.

- **📈 Interactive Analytics**  
  Visualize spending trends, budget progress, and category insights using **react-native-chart-kit**.

- **🌙 Adaptive Dark Mode**  
  Built with a custom **Theme Engine (Context API)** featuring a **Midnight Slate aesthetic**, neon sky-blue highlights, and glowing cards.

- **🎯 Smart Budgeting**  
  Set monthly financial goals with **live progress bars**, budget percentages, and **over-budget alerts**.
